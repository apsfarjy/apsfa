Library

LIBRARY RULES

  1. Books can be renewed only once at one given period provided there is no reservation for them.

  2. Each student can borrow 3 books at a time from the library and shall return the same within 7 days.

  3. Identity Card is compulsory for getting access to the library

  4. Silence to be maintained

  5. No discussion permitted inside the library

  6. Registration should be done to become a library member prior to using the library resources

  7. No personal belongings allowed inside the library

  8. Using Mobile phones and audio instruments with or without speaker or headphone is strictly prohibited in the library premises.

  9. Enter your name and Sign in the register kept at the entrance counter before entering library

  10. Show the books and other materials which are being taken out of the library to the staff at the entrance counter.

  11. Refreshment of any kind shall not be taken any where in the library premises.
